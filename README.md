# NightModePlus
NightModePlus est un script AutoHotKey qui vous permettra de baisser encore plus l'impression de luminosité de votre écran de PC même lorsque celle ci est déjà au minimum afin de préserver vos yeux. Pour cela, il créé un voile sombre devant vos applications.

Utilisation :
- Appuyez sur Ctrl + Alt + B pour activer le voile
- Appuyez sur Ctrl + Alt + H pour désactiver le voile

# English
NightModePlus is an AutoHotKey script that allows you to further reduce the perceived brightness of your PC screen—even when it’s already set to the minimum—to protect your eyes. It does this by placing a dark overlay in front of your applications.

How to use:
- Press Ctrl + Alt + B to enable the overlay
- Press Ctrl + Alt + H to disable the overlay
